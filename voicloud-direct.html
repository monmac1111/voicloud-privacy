プライバシーポリシー

施行日: 2026年6月10日

本ポリシーは、Mon Software が開発・提供する iOS アプリ「Voicloud Direct」(以下「本アプリ」) における情報の取り扱いについて説明するものです。私たちはお客様のプライバシーを保護し、すべての処理を透明かつ最小限のデータ収集で行うことをお約束します。

1. 収集するデータ

本アプリは以下のデータにアクセスします。これらはすべてお客様の iPhone 上にローカル保存され、お客様が明示的にアップロード・送信を選択しない限り、外部に送信されることはありません。

  • 音声録音 — マイクで録音された音声ファイルは、お客様の iPhone 内にローカル保存されます。
  • 文字起こし結果・要約 — 本アプリが生成・取得したテキストは、端末内のローカルストア (SwiftData) に保存されます。
  • 位置情報 (任意) — 録音時の場所をメタデータとして残す機能を ON にした場合のみ取得します。録音ファイルに付随して端末内に保存され、開発者には送信されません。いつでも OFF にできます。

2. 必要な OS 権限

  • マイク — 音声を録音するために必要です。
  • 位置情報 (任意) — 録音場所をメタデータとして残す場合のみ使用します。
  • カレンダー (任意) — カレンダーの予定からファイル名を自動生成する機能を使う場合のみ使用します。予定の情報は端末内でのファイル名生成にのみ使われ、外部には送信されません。

3. クラウドアップロード (任意)

お客様はご自身の iCloud Drive / Google Drive / Dropbox / OneDrive / Box アカウントに録音をアップロードできます。これらのアップロードはお客様ご自身のアカウントに直接保存され、開発者のサーバーを経由しません。OAuth 認証情報・トークンはお客様の端末の Keychain にのみ保存されます。

4. AI 機能 (文字起こし・要約 / BYOK)

本アプリの文字起こし・要約は「BYOK (Bring Your Own Key)」方式です。これらは任意機能であり、利用するにはお客様ご自身が取得した AI プロバイダの API キーを設定する必要があります。対応プロバイダは OpenAI、Google Gemini (Google AI Studio)、Google Vertex AI です。

音声・テキストの送信は、お客様が API キーを設定し、文字起こし・要約を実行したときのみ発生します。このとき、対象の録音音声またはそのテキストが、お客様の端末から、お客様が選択したプロバイダ (OpenAI または Google) の API へ直接送信されます。送信はお客様ご自身の API キーで認証され、Voicloud Direct がデータを中継することはありません。送信先プロバイダにおける課金・データの取り扱いは、各プロバイダの規約・プライバシーポリシーに従います。

API キーを設定しない場合、音声・テキストはいかなる AI サービスにも送信されません。録音・再生・共有シートでの書き出しといった主要機能は、API キーなしで動作します。

API キーはお客様の端末の Keychain に暗号化して保存され、開発者がアクセスすることはできません。開発者のサーバーに送信されることもありません。

5. 開発者サーバーへの送信なし

Voicloud Direct は、お客様の音声・文字起こし・要約・API キーを、開発者が所有または運営するサーバーに送信することはありません (そもそも当社は AI 推論サーバーを保有していません)。すべての処理はお客様の端末上、お客様が明示的に接続したクラウドサービス内、またはお客様が API キーを設定したうえで明示的に実行した AI サービス (opt-in) でのみ行われます。

6. アナリティクス・トラッキング

本アプリはアナリティクス SDK (Google Analytics、Firebase、Sentry 等) を一切使用していません。ユーザーの行動データ・使用統計を収集することも、トラッキングを行うこともありません。

7. 広告

本アプリは広告を表示しません。買い切り型のアプリで、広告 SDK を一切組み込んでいません。

8. お子様のプライバシー

Voicloud Direct は 13 歳未満のお子様を対象としたアプリではありません。当社は 13 歳未満のお子様から故意に個人情報を収集することはありません。

9. ポリシーの変更

本プライバシーポリシーは随時更新されることがあります。変更があった場合は、施行日を更新した上で本ページに反映されます。
