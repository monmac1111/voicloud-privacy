プライバシーポリシー

施行日: 2026年5月4日

本ポリシーは、Mon Software が開発・提供する Mac App Store アプリ「Voicloud for Mac」(以下「本アプリ」) における情報の取り扱いについて説明するものです。私たちはお客様のプライバシーを保護し、すべての処理を透明かつ最小限のデータ収集で行うことをお約束します。

1. 収集するデータ

本アプリは以下のデータにアクセスする場合があります。すべてのデータはお客様の Mac 上にローカル保存され、お客様が明示的にアップロード・送信を選択しない限り、外部に送信されることはありません。

  • 音声録音 — マイクおよびシステム音声 (ScreenCaptureKit 経由) で録音された音声ファイルは、お客様の Mac の Application Support / Library 内にローカル保存されます。
  • 文字起こし結果・要約・Action Items — Voicloud が生成したテキストはローカルの SwiftData ストアに保存されます。
  • ベクトル埋め込み — 関連文書サジェスト機能のため、Apple の NLEmbedding でローカル計算した数値ベクトルを録音と一緒に保存します。これは元の音声・テキストを再構成できるものではありません。

2. 必要な OS 権限

  • マイク — 音声 (お客様の声 + Zoom / Meet 等のシステム音声) を録音するために必要です。Voicloud は macOS 14.2+ の Core Audio Process Tap (CATap) を使ってシステム音声を取得するため、画面録画権限は不要です。
  • リマインダー (任意) — Action Items を macOS Reminders に送出する機能を使う場合のみ必要です。

3. クラウドアップロード (任意)

お客様はご自身の Google Drive / Dropbox / OneDrive / Box / iCloud Drive アカウントに録音をアップロードできます。これらのアップロードはお客様のアカウントに直接保存され、開発者のサーバーを経由しません。OAuth 認証情報は macOS Keychain にのみ保存されます。

4. AI 機能 (Apple Intelligence + 任意の Gemini AI)

デフォルト (オフライン): 文字起こしは Whisper.cpp、要約・Action Items 抽出・関連文書サジェストは Apple Intelligence (FoundationModels) を使用し、すべてお客様の Mac 上で完結します。

opt-in (Gemini AI): お客様が Settings で Gemini API キー (Google AI Studio または Vertex AI) を入力した場合のみ、文字起こしや要約のために音声・テキストを Google のサービスに送信します。この場合、Google 側で別途請求が発生します。Voicloud は Google にデータを中継することはなく、お客様のデバイスから直接 Google API にリクエストが送られます。

AI 自動タイトル (Gemini Flash): 文字起こし完了後、Gemini Flash 2.5 が文字起こしテキストを読み、内容に合った題名を提案します。送信されるのは録音のテキスト (音声ファイルではない) のみで、お客様の Settings で「AI 自動タイトル」を OFF にした場合は送信されません。ユーザーが手動で題名を編集した録音についてはタイトル変更を行いません。

API キーは macOS Keychain に暗号化されて保存され、Voicloud の開発者がアクセスすることはできません。

4-2. 通知センター

Voicloud は録音保存・文字起こし完了・要約完了・パイプライン失敗を macOS の通知センターに配信します。通知の内容 (録音タイトル等) はお客様の Mac 上で生成され、Apple 経由で送信されるのは通知ペイロード (通知タイトルと本文) のみです。これらは Apple のローカル通知システム経由で配信され、外部サーバーへの送信はありません。各イベントは Settings から個別に ON/OFF できます。

5. 開発者サーバーへの送信なし

Voicloud は、お客様の音声・文字起こし・要約・Action Items データを開発者が所有または運営するサーバーに送信することはありません。すべての処理はお客様の Mac 上、またはお客様が明示的に許可したクラウドサービス内 (Google Drive など) または AI サービス (Gemini API、opt-in) で行われます。

6. 診断データ (MetricKit)

本アプリは Apple 純正の MetricKit を介して、クラッシュ・ハング・メモリ使用量・CPU 例外などの診断情報を収集する場合があります。これらのデータは Apple が集約し、デバイス上の Voicloud 専用フォルダ (Application Support/Voicloud/Diagnostics) に JSON として保存されます。Apple のシステム設定 → プライバシーとセキュリティ → 解析と改善 → 「App デベロッパと共有」がオフの場合、開発者には一切送信されません。

7. アナリティクス

本アプリはアナリティクス SDK (Google Analytics、Firebase、Sentry 等) を一切使用していません。ユーザーの行動データ・使用統計を独自に収集することはありません。

8. 広告

本アプリは広告を表示しません。Voicloud は買い切り型のアプリで、広告 SDK を一切組み込んでいません。

9. お子様のプライバシー

Voicloud は 13 歳未満のお子様を対象としたアプリではありません。当社は 13 歳未満のお子様から故意に個人情報を収集することはありません。

10. ポリシーの変更

本プライバシーポリシーは随時更新されることがあります。変更があった場合は、施行日を更新した上で本ページに反映されます。重大な変更があった場合は、アプリ内通知でお知らせします。
